## 项目具体参数

根据招标公告附件1，本项目（包72：国网冀北电科院2026年基于人工智能平台的绿色低碳采购辅助智能体构建-开发项目）参数如下：

| 参数 | 值 |
|------|-----|
| **价格评分方法** | 区间平均价浮动法 |
| **权重比例** | 技术:商务:价格 = **60:10:30** |
| **n1** | 1 |
| **n2** | 0.5 |
| **w1** | -0.15 |
| **w2** | 0.1 |
| **C值** | 随机抽取，范围 [0, 3%]，级差 0.5% |

## 区间平均价浮动法 详细计算步骤

### 步骤1：确定有效投标人

| 合格投标人数量（M） | 剔除规则 |
|-------------------|---------|
| M ≤ 5 | 无剔除 |
| 5 < M ≤ 10 | 去掉最高和最低各1个 |
| 10 < M ≤ 20 | 去掉最高、次高和最低各1个 |
| 20 < M ≤ 30 | 去掉最高、次高、第三高和最低、次低 |
| M > 30 | 去掉4个较高 + 3个较低 |

**有效投标价范围**：剔除后剩余投标人的评标价算术平均值 A1 的 **[1+w1, 1+w2] 区间**

- w1 = **-0.15**（即平均值的 85%~100%）
- w2 = **0.1**（即平均值的 100%~110%）

> 注：区间为**不含本数**，即 (0.85A1, 1.1A1)

**特殊情况**：若区间内合格投标人数为0，则所有初评合格投标人均为有效投标人

---

### 步骤2：计算基准价

1. 计算有效投标人的评标价算术平均值 **A2**
2. **基准价 = A2 × (1 - C)**
   - C 为基准价浮动系数，**开标时随机抽取**
   - 抽取范围：[0, 3%]，级差 0.5%
   - 即 C 可能是：0%, 0.5%, 1%, 1.5%, 2%, 2.5%, 3%

---

### 步骤3：计算价格得分

$$\text{价格部分得分} = 100 - 100 \times n \times \frac{|\text{评标总价} - \text{基准价}|}{\text{基准价}}$$

| 评标总价与基准价比较 | n取值 |
|-------------------|-------|
| 评标总价 **≥** 基准价 | **n = n1 = 1** |
| 评标总价 **<** 基准价 | **n = n2 = 0.5** |

> **说明**：高价高于基准价扣分力度大（n1=1），低价低于基准价扣分力度小（n2=0.5）
> 
> **注意**：计算结果小于0时，按0分计

---

## 举例说明

假设：
- 7家投标人通过初步评审（5 < M ≤ 10）
- 去掉最高和最低后剩余5家有效投标人
- 5家有效投标人的评标价：160万、165万、170万、175万、180万
- A1 = (160+165+170+175+180)/5 = 170万
- 有效报价区间 = (170×0.85, 170×1.1) = (144.5, 187)
- 5家报价都在区间内，都是有效投标人
- A2 = 170万
- 假设开标抽取 C = 1.5%
- 基准价 = 170 × (1 - 0.015) = 167.45万

| 投标人 | 评标总价 | 与基准价比较 | n   | 价格分            |            |                     |
| --- | ---- | ------ | --- | -------------- | ---------- | ------------------- |
| A   | 165万 | <基准价   | 0.5 | 100 - 100×0.5× | 165-167.45 | /167.45 = **99.27** |
| B   | 167万 | <基准价   | 0.5 | 100 - 100×0.5× | 167-167.45 | /167.45 = **99.87** |
| C   | 170万 | >基准价   | 1   | 100 - 100×1×   | 170-167.45 | /167.45 = **98.48** |
| D   | 175万 | >基准价   | 1   | 100 - 100×1×   | 175-167.45 | /167.45 = **95.49** |

## ⚠️ 低于成本价认定

在评标过程中，若投标报价存在以下情况，可能被认定为"低于成本"：

1. **投标报价 < 全部通过符合性审查供应商投标报价平均值 × 50%**
2. **投标报价 < 通过符合性审查的次低报价 × 50%**
3. **投标报价 < 采购项目最高限价 × 45%**
4. **评审委员会基于专业判断，认为报价过低可能影响履约**

被要求澄清的投标人需提供：
- 项目具体成本测算
- 报价合理性书面说明
- 必要证明材料

**未按规定时间回复者，视为无效**

## 综合评分计算

**总分 = 技术得分 × 60% + 商务得分 × 10% + 价格得分 × 30%**

综合评分相等时：
1. 技术得分高者优先
2. 技术得分相同，投标报价低者优先
3. 都相同，由招标人确定
