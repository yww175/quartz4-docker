# 商务评分标准（FWSW01：服务类通用）

> 来源：招标文件第三章 评标办法前附表之三
> 招标编号：JB26-QTFW-ZB02-0126L2

---

## 1. 诚信评价（0分）

### （1）不良行为处理情况通报（-30~0分）

| 情形 | 扣分 |
|-----|-----|
| 不存在不良行为；或解除距投标截止日超过3年 | **不扣分** |
| 解除距投标截止日超过2年但在3年以内（含） | **扣5分** |
| 其中存在行贿行为的 | **扣10分** |
| 解除距投标截止日超过1年但在2年以内（含） | **扣10分** |
| 其中存在行贿行为的 | **扣20分** |
| 解除距投标截止日1年以内（含） | **扣15分** |
| 其中存在行贿行为的 | **扣30分** |

> 依据：《国家电网有限公司供应商关系管理办法》中的不良行为处理情况

### （2）公共信用信息报告、企业信用信息公示报告（-10~0分）

| 情形 | 扣分 |
|-----|-----|
| 不存在失信行为，或质量、诚信以外失信行为，情节轻微，罚款累计1万元以下 | **不扣分** |
| 质量、诚信以外失信行为，罚款累计1万元及以上~30万元以下 | **扣5分** |
| 存在质量、诚信失信行为，或罚款累计30万元及以上（未纳入失信黑名单） | **扣10分** |

### （3）影响招投标工作公正性行为的凭证（-20~0分）

| 情形 | 扣分 |
|-----|-----|
| 投标截止日近一年内，存在打招呼、请托关照等行为 | **扣20分** |
| 无上述行为 | **不扣分** |

## 2. 绿色低碳评价（3分）

### 绿电绿证（0~3分）

| 情形 | 得分 |
|-----|-----|
| 取得绿色电力证书或凭证 | **3分** |
| 未取得 | **0分** |

---

## 3. 科研创新评价（7分）

### （1）企业创新成果（0~1分）

| 情形 | 得分 |
|-----|-----|
| 取得过国家级科技创新成果 | **1分** |
| 取得过省级科技创新成果 | **0.5分** |
| 取得过其他级别创新成果 | **0.3分** |
| 未取得任何级别创新成果 | **0分** |

> 评价期：投标截止日近3年内

### （2）创新激励机制（0~1分）

| 情形 | 得分 |
|-----|-----|
| 企业制定创新激励相关政策和机制 | **1分** |
| 企业未制定 | **0分** |

### （3）研发团队规模（1~3分）

| 情形 | 得分 |
|-----|-----|
| 高级及以上职称人员 + 高级技师 ≥30人 | **3分** |
| 高级及以上职称人员 + 高级技师 ≥15人 | **2分** |
| 高级及以上职称人员 + 高级技师 ＜15人 | **1分** |

### （4）科研经费占比（0~2分）

| 情形 | 得分 |
|-----|-----|
| 任一年研发投入占比达2%以上 | **2分** |
| 有研发投入（不足2%） | **1分** |
| 无研发投入 | **0分** |

> 评价期：投标截止日近3年内

---

## 4. 综合评价（90分）

### （1）综合实力（26~36分）

对投标人的能力、人员、资质等综合实力进行评审

| 等级 | 得分 |
|-----|-----|
| 优 | **32-33分** |
| 良 | **29-31分** |
| 一般 | **26-28分** |

**加分项：**
- 获得中国质量奖、中国质量奖提名奖、全国质量奖及中国工业大奖任意一项及以上 → **加2分**
- 具有高新技术企业证书 → **加1分**

### （2）投标响应（41~54分）

对商务投标文件的整体情况进行评审（格式完整性、目录编制、偏差表、遗漏项数、文件签署等）

| 等级 | 得分 |
|-----|-----|
| 优：提交完整且整体情况优良 | **22-24分** |
| 良：提交基本完整且整体情况较好 | **19-21分** |
| 一般：存在多项缺漏项或整体情况一般 | **16-18分** |

### （3）报价质量（25~30分）

对投标报价完整性、准确性、规范性、合理性进行综合评审

| 等级 | 得分 |
|-----|-----|
| 优 | **29-30分** |
| 良 | **27-28分** |
| 一般 | **25-26分** |

---

## 注意事项

1. 本评分标准适用于 **FWSW01：服务类通用**
2. 各标包选用哪种评分标准详见招标公告附件1
3. 评价中涉及的"投标截止日近X年"均指投标截止日前X年
